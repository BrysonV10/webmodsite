Get Started

Install the WebMod extension here, then check out our Script Store to find an appealing script to satisfy your needs

How to use

Find the script that you want to install then hit download. If you do not have the WebMod Extension the wheel will spin forever but if you do have it then it will come up with download button. Hit the download button and the script will be automatically install and ready to use

What does WebMod do?

WebMod is an easy to use userscript extension that allows you to edit any websites you visit (Except a new tab and built in sites) by using JavaScript. If you would like to learn JavaScript then go to our GitHub page here
